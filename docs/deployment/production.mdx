## Production Checklist

<AccordionGroup>
  <Accordion title="Security" icon="shield">
    - [ ] Enable HTTPS with valid SSL certificate
    - [ ] Use strong passwords for all services
    - [ ] Enable firewall (allow only 80, 443, 22)
    - [ ] Disable debug mode
    - [ ] Use secrets management
    - [ ] Enable database encryption
    - [ ] Configure Redis password
  </Accordion>
  
  <Accordion title="Performance" icon="gauge">
    - [ ] Enable database connection pooling
    - [ ] Configure Redis caching
    - [ ] Set up CDN for static files
    - [ ] Enable gzip compression
    - [ ] Configure worker concurrency
    - [ ] Set up database indexes
  </Accordion>
  
  <Accordion title="Monitoring" icon="chart-line">
    - [ ] Set up application monitoring
    - [ ] Configure log aggregation
    - [ ] Enable error tracking
    - [ ] Set up uptime monitoring
    - [ ] Configure alerting
  </Accordion>
  
  <Accordion title="Backup" icon="database">
    - [ ] Automated daily database backups
    - [ ] Off-site backup storage
    - [ ] Test restore procedures
    - [ ] Document recovery process
  </Accordion>
</AccordionGroup>

## Nginx Configuration

```nginx
server {
    listen 80;
    server_name vmledger.example.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name vmledger.example.com;
    
    ssl_certificate /etc/letsencrypt/live/vmledger.example.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/vmledger.example.com/privkey.pem;
    
    # Frontend
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
    
    # API
    location /api {
        proxy_pass http://localhost:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

## SSL Certificate

```bash
# Install Certbot
sudo apt install certbot python3-certbot-nginx

# Get certificate
sudo certbot --nginx -d vmledger.example.com

# Auto-renewal
sudo certbot renew --dry-run
```

## Scaling

### Horizontal Scaling

```yaml
# docker-compose.prod.yml
services:
  api:
    deploy:
      replicas: 3  # Run 3 API instances
  
  celery-worker:
    deploy:
      replicas: 5  # Run 5 worker instances
```

### Vertical Scaling

```yaml
services:
  api:
    deploy:
      resources:
        limits:
          cpus: '2'
          memory: 4G
```

## Monitoring

### Prometheus + Grafana

```yaml
services:
  prometheus:
    image: prom/prometheus
    ports:
      - "9090:9090"
    volumes:
      - ./prometheus.yml:/etc/prometheus/prometheus.yml
  
  grafana:
    image: grafana/grafana
    ports:
      - "3001:3000"
    depends_on:
      - prometheus
```
